Place bundled FFmpeg binaries here before building.

Expected filenames:
- ffmpeg-windows-x64.exe
- ffprobe-windows-x64.exe
- ffmpeg-macos-x64
- ffprobe-macos-x64
- ffmpeg-macos-arm64
- ffprobe-macos-arm64
- ffmpeg-linux-x64
- ffprobe-linux-x64

On macOS/Linux, ensure the binaries are executable:
- `chmod +x ffmpeg-macos-* ffprobe-macos-* ffmpeg-linux-x64 ffprobe-linux-x64`
